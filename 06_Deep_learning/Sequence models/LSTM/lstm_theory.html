LSTM: Long Short-Term Memory

A gated RNN architecture for handling long-term dependencies in sequential data

⬡ 1. What is LSTM?
LSTM stands for Long Short-Term Memory. It is an improved version of RNN that is designed to remember important information for a long time in sequential data.

Main Idea

  • LSTM processes data step by step like RNN.
  • It has a special memory system.
  • It decides what to remember, forget, and output.

Used For

  • Text classification
  • Sentiment analysis
  • Speech recognition
  • Machine translation
  • Time-series prediction

Core Strength

  • Handles long sequences better than simple RNN.
  • Preserves long-term context.
  • Reduces vanishing gradient problem.
⬡ 2. Why do we need LSTM?

Simple RNNs struggle when important information appears far behind in the sequence.

Problems in Simple RNN

  • Difficulty in learning long-term dependencies.
  • Vanishing gradient problem.
  • Exploding gradient problem.
  • Old context may be overwritten by new words.
  • Long sequences become hard to remember.

Reason

  • RNN mainly uses one hidden state.
  • The same path carries both short-term and long-term information.
  • As sequence length increases, earlier information becomes weak.
Important correction: LSTM does not have two hidden states. It has one hidden state ht and one cell state Ct. The cell state behaves like long-term memory, while the hidden state behaves like short-term or current context.
⬡ 3. Intuition: Long-Term Dependency

Consider this sentence:

Vinod is a student of B.Tech final year. He studies at VNIT Nagpur.

Short-Term Context

  • Current words around the present time step.
  • Example: while reading “He studies”, the current local context is “He studies”.

Long-Term Context

  • Information from an earlier part of the sequence.
  • Example: “He” refers to Vinod.
  • The model must remember that Vinod is male.

LSTM Role

  • Stores useful information in the cell state.
  • Carries important context forward.
  • Uses gates to control memory flow.

⬡ 5. Cell State vs Hidden State

Cell State Ct

  • Acts like long-term memory.
  • Carries useful information across many time steps.
  • Information flows through it with fewer transformations.
  • Helps reduce the vanishing gradient problem.

Hidden State ht

  • Acts like short-term or current memory.
  • Represents the output of the current time step.
  • Passed to the next LSTM cell.
  • Used for prediction or further sequence processing.
Simple memory intuition: Ct remembers important long-term information. ht focuses on the current output and recent context.
⬡ 6. Three Gates of LSTM

LSTM has three main gates. Each gate controls the flow of information.

1. Forget Gate

Question: What old information should I forget?

Purpose: Removes useless information from previous cell state.

Range: 0 to 1 using sigmoid.

2. Input Gate

Question: What new information should I store?

Purpose: Adds useful new information to cell state.

Uses: Sigmoid + tanh.

3. Output Gate

Question: What should I output now?

Purpose: Creates current hidden state ht.

Uses: Sigmoid and updated cell state.

⬡ 12. Important Equations
Part Equation Meaning
Forget Gate ft = sigmoid(Wf[ht-1, xt] + bf) Decides what to forget from old cell state.
Input Gate it = sigmoid(Wi[ht-1, xt] + bi) Decides what new information to store.
Candidate Cell State C̃t = tanh(Wc[ht-1, xt] + bc) Creates possible new memory.
Cell State Update Ct = ft × Ct-1 + it × C̃t Combines old useful memory and new useful memory.
Output Gate ot = sigmoid(Wo[ht-1, xt] + bo) Decides what should be output.
Hidden State ht = ot × tanh(Ct) Creates current output or short-term memory.
⬡ 13. Simple Real-Life Analogy

Imagine you are reading a paragraph. You do not remember every word equally. Your brain continuously decides:

Forget

  • Remove useless or old information.
  • Similar to forget gate.

Remember

  • Store important new information.
  • Similar to input gate.

Use

  • Use relevant memory to understand current word.
  • Similar to output gate.
One-line intuition: LSTM works like a smart memory system that decides what to forget, what to remember, and what to output.

⬡ 15. Final Summary

Definition

  • LSTM is a gated RNN architecture.
  • It is designed to handle long-term dependencies.
  • It uses memory cells to preserve useful information.

Key Components

  • Cell state Ct
  • Hidden state ht
  • Forget gate
  • Input gate
  • Output gate

Main Benefit

  • Remembers important context for a longer time.
  • Handles long sequences better than RNN.
  • Reduces vanishing gradient problem.
Best one-line definition: LSTM is a sequential neural network architecture that improves RNN by using gated memory cells to preserve long-term context and handle long sequences more effectively.
